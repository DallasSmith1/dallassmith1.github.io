My Transit

My Transit is a mobile application for GO Transit, built using Flutter. It leverages the GO Transit API to deliver comprehensive transit data, offering users easy navigation and access to valuable information.

Key features include:

  • Saving and swapping trips for easy access
  • Live GPS tracking
  • Platform numbers (available up to 1 hour before station screens)
  • Scheduled and computed departure times
  • Quick access to purchase E-Tickets
  • Alerts affecting the current trip

The app also provides detailed information about all GO Transit alerts, ensuring users stay informed about their travel plans.
